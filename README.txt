Project Information:
    Name: Poetry Clock
    Func: It's a clock help you to work more efficently and healthy, and when the time run out, you can hear some wonderful poetry.
    Ho2U: run the python script by " python clock.py duration ", where the duration is a number as the seconds you want to last.
          If there is no any duration parameter, the program will set duration as 2400 (40 mins).
	  python clock.py [-option] duration
	  -option:
		-p: set the poetry as the clock
		-w: set the web as the clock
	  duration:
		should be integer as second
